# Enhancements by [almightymoon](https://github.com/almightymoon)

**NovaMart** — a futuristic 3D e-commerce storefront forked from [NZLouislu/Shop3D](https://github.com/NZLouislu/Shop3D).

## What's different

- **Dark cyberpunk UI** — Cyan/purple theme, glass navbar, glowing product cards
- **Hero banner** — "Shop in full 3D" landing with stats and CTAs
- **360° badges** — Product cards highlight interactive 3D viewing
- **Enhanced 3D viewer** — Night environment, faster auto-rotate, neon backdrop
- **NovaMart branding** — Replaced upstream branding throughout
- **Docker** — Multi-stage Next.js standalone container
- **GitHub Actions CI** — Lint + production build

## Quick Start

```bash
npm install
npm run dev          # http://localhost:3000

docker compose up --build
```
